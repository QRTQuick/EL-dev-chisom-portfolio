# EL-Dev Chisom - Portfolio Project

A professional developer portfolio showcasing multi-disciplinary software engineering expertise.

## Tech Stack
- **Frontend**: React 18 + TypeScript + Tailwind CSS
- **Build Tool**: Vite
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Deployment**: Vercel/Netlify

## Project Structure
```
portfolio/
├── src/
│   ├── components/
│   ├── sections/
│   ├── data/
│   └── styles/
├── public/
└── package.json
```

## Development
```bash
npm install
npm run dev
```
